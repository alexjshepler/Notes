## Control Structure

Selection Statements

- If else
  - If true do this
  - Else do this
- Switch
  - Given input do associated task

Loops

- For
  - Loop for a set amount of time
- While
  - Loop while a condition is true
- Do while
  - Run through the loop and then check the condition, if true run again, else continue
